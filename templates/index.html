How EMI is Calculated

EMI = [P × r × (1+r)^n] / [(1+r)^n − 1]

  • P = Principal Loan Amount
  • r = Monthly Interest Rate (Annual Rate / 12 / 100)
  • n = Loan Tenure in Months

ℹ️ About This System

Understanding the AI Loan Approval Engine

🤖 Machine Learning Model

This system uses a Random Forest Classifier trained on 2,000 synthetic loan applications. The model evaluates multiple factors including income, credit score, employment history, and debt-to-income ratios to predict loan default risk.

📋 Approval Rules

  • Age: Must be between 18–59 years
  • Credit Score (Age 18–27): Must be 100–450
  • Credit Score (Age 28+): Must be 500–900
  • EMI Limit: Monthly EMI must not exceed 50% of income
  • ML Risk: Model must classify applicant as "Low Risk"
